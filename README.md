This is a remade version of the old Tcp chat app, made in Rust and using BekBraces youtube tutorial (https://www.youtube.com/watch?v=653rafFNBmA). 
The purpose of this was to learn how to use cursive to create custom UIs for other projects.

The program itself operates much like the older project
Server is run using cargo run --bin server
Client is run using cargo run --bin client 'username'
